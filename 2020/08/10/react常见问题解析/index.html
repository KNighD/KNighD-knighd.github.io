Q：为什么 react 组件开头需要大写？

A：React.createElement 的第一个参数表示类型。如果开头小写，类型为字符串，此时 react 会认为是一个原生的 dom 节点。而如果是大写，则类型会是个变量，react 会认为这是个自定义的组件：
